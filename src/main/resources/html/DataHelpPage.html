Data Help Page

Data page info and functionality:

The Data page displays the raw data about the charging stations in a table.

Sorting The Table

Clicking on the attributes at the top of table will sort the data according to the attribute clicked.

Station Information

Clicking on a station in the table will display the full information about that station in the textbox in the sidebar.

Add / Edit Station

Signed in users are able to add / edit station data.
