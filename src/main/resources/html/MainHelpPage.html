Help Page

Gecko:

Gecko is an app designed for electric car user, they can plan trips between charging stations and save these journeys for future use.

Click the buttons on the left for more specific help pages about the application.

Importing Data

The first time a user loads the application, the station data needs to be imported.
The imported data needs to be the form of a csv file.
Clicking the import data button in the bottom left will ask the user to select a file.
