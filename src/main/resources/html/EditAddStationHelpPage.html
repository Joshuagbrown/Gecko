Add/Edit Station Help Page

Add / Edit station functionality:

To Add / Edit station data, you initially have to be signed in

Editing from Popup

Editing from a popup is as simple as initially clicking on a station and then clicking the edit station button. Information that is edited will only be displayed per user.

Editing from Side Bar

Clicking on the station, information will appear on the left. You can edit information from the button shown on the left.

Add Station

To add a station, you can add from popup, or sidebar. Adding the station you will need the Name, Address, Operator, Owner, Carpark count, and time limit.
You can also check whether it has an Attraction, Charging cost, is 24 Hours, and Carpark cost.

Adding from Popup

To add a station from marker, click on your specified location on the map, and then click add station. The address will be auto filled.

Adding from Side Bar

To add a station from sidebar, click the add station button on the left of the screen.
