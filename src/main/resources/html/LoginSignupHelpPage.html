Login / Signup Help Page

Login / Signup page info and functionality:

The Login / Signup page allows for users to login to the app.
Logging into the app allows for users to register vehicles, add / edit stations, and save journeys.
Click the login button in the top right to login.

Creating an Account:

The signup page is used to create an account. You will need to create a username, password, and set your home address.
After doing this, and signing in, user specific options will be avalible to you. These include:
Registering vehicles, editing station data, and saving joruneys.

Username:

Please enter your Username, the username must be unique, and have no special characters.

Password:

Please enter your Password, must be atleast 8 characters.

Personal Name

When signing up, you will need to enter a personal name.

Home Address

When signing up, you will need to enter a valid home address.

Signup Button

This will signup and log you in.

Login Button

To login, enter registered username and password. Click 'Login' this will log you in

Logout

A user can logout at any time by going to 'My Details' page, and clicking the logout button.

Fun Facts (Top Right)

Random fun facts will be displayed each time you go to log in
