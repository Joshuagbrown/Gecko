Map Help Page

Map page info and functionality:

The map page displays a map and a toolbar with functions and a textbox.
On the map all the chargings station in the database will be shown with 'Green Markers'.
Clicking a station on the Map will display the full information about that station in the text box in the sidebar.

Plan a Trip:

This function will show a route on the map with instructions from the start location to the end location given.
To plan a trip enter a start location and an end location into the text fields either by searching an address or clicking on the map and selecting autofill.

Once done click the 'Go!' button and the map will display the route from the start location to the end location.
These markers at the start and end locations are also draggable so feel free to move it around and the route and instructions will change accordingly.
To add a stop simply click the 'Add Stop' button and enter in the address on the stop and click the 'Go!' button again to apply that new route. A maximum of 5 stops can be added.


To remove the route click the 'Remove Route' button and this will remove the route from the map and also reset the text boxes.
A save journey button is available to signed in users.

Legend:

There is a collapsable legend on the bottom left of the map to display what each symbol on the map means
