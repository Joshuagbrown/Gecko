Login Help Page

Login page info and functionality:

The login page allows for users to login to the app.
Logging into the app allows for users to rate, share, and save routes.
Click the login button in the top right to login.

Creating an Account:

The signup page is used to create an account. You will need to create a username, password, and set your home address.
After doing this, and signing in, user specific options will be avalible to you. These include:
Editing station data, rating stations, and saving routes.
