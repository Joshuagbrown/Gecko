Filter Help Page

Filter functionality:

The Filter Station function:

This function will hide the stations in the table that don't fit the filters given.
To filter the stations in the table (This will apply on both the map and data page) the options are:

Search with station name:

This will hide all the stations that don't have the input as either the address or name.

Distance (Kilometer):

This will hide all the stations that are outside the distance given from the slider from a point selected on the map or home address if signed in..

Time Limit (Minutes):

This will hide all the stations that have a time limit a below the value given from the slider.

Is 24 Hour:

This will hide all the stations that aren't available 24 hours.

No Carpark Cost:

This will hide all the stations that have a carpark cost.

No Charging Cost:

This will hide all the stations that have a charging cost.

Has Tourist Attraction:

This will hide all the stations that don't have a tourist attraction.

Once selected, clicking the 'Filter' button will apply those filters.
To reset back to all the stations just click the 'Reset Filter' button.
